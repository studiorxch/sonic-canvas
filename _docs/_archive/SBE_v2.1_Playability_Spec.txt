SBE v2.1 — PLAYABILITY PATCH (SPEC)

OBJECTIVE
Improve usability and musical control without expanding UI surface.

Focus:
- inspector cleanup
- MIDI control (grid + quantize)
- ball system (radius separation + spawn + swarm)
- UI compression

DO NOT:
- change physics engine structure
- change collision logic core
- introduce frameworks

---

1. INSPECTOR CLEANUP

REMOVE:
- “Text”, “text-1”, “editing…” labels
- “Create / Update” buttons
- group controls
- redundant descriptive text

NEW STRUCTURE:

[ CONTENT ]

[ X ] [ Y ] [ CENTER ]

[ size ] [ scale ] [ rotation ]

--- MIDI ---
[ NOTE GRID ]

--- BEHAVIOR ---
type
strength

--- APPEARANCE ---
color
outline

RULES:
- no unnecessary labels
- no scroll required
- compact layout only

---

2. PLAYER BAR

KEEP:
- BPM (input only)
- Start / Stop
- Reset
- Demo (rename Load → Demo)
- Open / Save
- Font

CONVERT TO ICONS:
▶ ⏹ ⟲ ⧉ 📂 💾 T

STYLE:
- opacity ~0.8
- backdrop blur
- reduced padding

---

3. MIDI SYSTEM

REMOVE:
- dropdown selector
- +/- buttons

ADD GRID:

C  C# D  D# E
F  F# G  G# A
A# B

BEHAVIOR:
- click assigns note
- highlight active
- auto color

RESTORE:
- quantize toggle
- divisions (1/4, 1/8, 1/16)

Quantize affects MIDI timing only.

---

4. BALL SYSTEM

REPLACE:
radius

WITH:
collisionRadius
renderRadius

RULE:
collisionRadius = physics
renderRadius = visuals

DEFAULT:
collisionRadius = 6
renderRadius = 12–20

STYLE:
"solid" or "core"

Core:
- outer ring = renderRadius
- inner core = collisionRadius * 0.6

---

5. BALL TOOL

FIX:
- spawn exactly at cursor
- drag sets velocity

ADD MINI PANEL:
Count
Speed
Spread

---

6. COLLISION FIX

if collisionCount > threshold:
  apply random velocity

if overlap:
  push objects apart

---

7. DRAW TOOL

CHANGE:
- click = point
- drag = curve
- double click = finish

REMOVE:
- Finish Path button

---

8. TERMINOLOGY

Thickness → Outline

---

9. ENVIRONMENT

ADD:
- background image loader
- transparent background toggle

---

10. PRESENTATION MODE

FIX:
- hide UI
- disable editing
- keep simulation running

---

11. TYPOGRAPHY

USE:
- Inter
- Helvetica Neue

REMOVE:
- serif fonts

---

12. PERFORMANCE

REMOVE:
- heavy glow effects

LIMIT:
- ball count
- velocity spikes

---

ACCEPTANCE CRITERIA

- inspector fits screen
- no redundant UI text
- MIDI grid replaces dropdown
- quantize works
- balls scale visually without affecting rhythm
- precise spawn
- presentation mode works
- clean typography

---

RESULT

Playable → Predictable → Controllable
